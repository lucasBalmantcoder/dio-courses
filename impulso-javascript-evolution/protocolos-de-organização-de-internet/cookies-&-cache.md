1. Cookies
   Os Cookies são pequenos arquivos de texto enviados pelo servidor para o navegador do usuário. Eles servem primordialmente para identificar o usuário e armazenar informações sobre suas preferências e sessões.

Principais Finalidades:
Gerenciamento de Sessão: Mantém o usuário logado em um site sem a necessidade de redigitar a senha a cada nova página.

Personalização: Armazena preferências configuradas pelo usuário, como idioma, tema (claro/escuro) ou itens em um carrinho de compras.

Rastreamento (Tracking): Utilizado por serviços de análise e marketing para registrar o comportamento do usuário e exibir anúncios direcionados.

2. Cache
   O Cache do navegador é um sistema de armazenamento de arquivos estáticos. Em vez de baixar os mesmos arquivos toda vez que você visita um site, o navegador guarda uma cópia local para acelerar o carregamento.

Principais Finalidades:
Velocidade de Carregamento: Reduz drasticamente o tempo de abertura de páginas já visitadas, pois os arquivos são lidos diretamente do disco rígido ou memória RAM do computador.

Economia de Banda: Evita o download repetitivo de arquivos pesados, como imagens de alta resolução, arquivos CSS e scripts JavaScript.

Melhoria na Experiência (UX): Torna a navegação mais fluida e responsiva.
